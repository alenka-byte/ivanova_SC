
**Задание:**  
Реализовать моделирование изменения температуры в пластине на основе одномерного уравнения теплопроводности с использованием метода конечных разностей.

Начальные параметры:
* Плотность тела 19300 кг/м$^2$
* Удельная теплоемкость 130 Дж/(кг°C)
* Коэффициент теплопроводности 317 Вт/(м°C)
* Модельное время 2 с
* Температура справа 100°C
* Температура слева 20°C
* Температура пластины 50°C
* Ширина пластины 0,02 м


| Шаг по пространству, м \ Шаг по времени, с | 0.1      | 0.01     | 0.001    | 0.0001   |
| ------------------------------------------ | -------- | -------- | -------- | -------- |
| 0.1                                        | не число | не число | не число | не число |
| 0.01                                       | 59,5582  | 59,7280  | 59,7430  | 59,7444  |
| 0.001                                      | 59,7744  | 59,8891  | 59,8981  | 59,8990  |
| 0.0001                                     | 59,7761  | 59,8902  | 59,8992  | 59,9001  |
**Вывод**
Так как ширина пластины 0,02 м меньше шага по пространству 0,1 м, то невозможно посчитать температуру, поэтому выводится "не число"
